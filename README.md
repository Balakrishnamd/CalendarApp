# React Calendar Pro

A polished month-view calendar (React + Vite + Tailwind + dayjs).

## Features
- Shows current month by default with grid layout
- Navigate previous/next months, quick "Today" button
- Highlights today's date
- Loads events from a static `/public/events.json` (title, date, time, duration, color)
- Handles overlapping events by assigning visual "lanes" and indicating conflicts
- Clean, responsive UI

## Quickstart
```bash
npm install
npm run dev
```
